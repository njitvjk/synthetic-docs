# Synthetic Invoices & Contracts Generator

This repository generates **100 synthetic PDF files** (50 invoices + 50 contracts) containing realistic fake PII for automation testing and research reproducibility.

## Features
- Uses **Faker** for generating synthetic people, company, and address data.
- Outputs standardized **PDF documents** suitable for workflow automation testing.
- 100% fake data (no real PII).
